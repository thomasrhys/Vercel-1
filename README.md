# Vercel-1

This project is a demonstration of deployment on Vercel.

## Getting Started

To get started with this project, follow the steps below:

1. Clone the repository:
   ```bash
   git clone https://github.com/thomasrhys/Vercel-1.git
   ```

2. Install dependencies:
   ```bash
   cd Vercel-1
   npm install
   ```

3. Run the project:
   ```bash
   npm run dev
   ```

## Features
- Easy deployment to Vercel
- Modern JavaScript Support
